# SheCodesPsy — Reprogramming stereotypes! 
## 13-15 May 2026, Granada, Spain. 

Welcome to **SheCodesPsy — Reprogramming Stereotypes**! workshop. This is a 3-day immersive course designed for psychology students who want to learn programming while exploring its relevance for psychological research. No prior coding experience needed! 

This guide will walk you through the course requirements and the different open-acccess materials that we'll use across the sessions.

---

## Requirements

To follow the workshop you will need:

- A **laptop or tablet** with a browser installed (Google Chrome or Mozilla Firefox recommended)
- Access to internet: **Eduroam connection** will work in all the University of Granada installations. 
- A **Google account** so you can use and store the Google Colab notebooks that we'll work with. 

> **OPTIONAL**: *For those who want to go to the bottom of it,* you can also install **PsychoPy** on your computer (instructions at the end of this document). Psychopy is a python-based library for experiment presentation. Having it installed is **not mandatory** for the worksho: we will not do exercises with it. Still, we will follow a quick demo to show how to code a basic experiment. If you'd like to follow along and try the experiment yourself. it's worth setting up.

---

## What we will cover

- **Python basics**: variables, data types, control flow: conditionals, loops.
- **Functions & libraries**: working with tools built for research
- **Data handling & analysis**: intro to real Psychology-style data
- **Project-based teamwork**: collaborative coding in groups to achieve meaningful, psychology-related goals. 
- **Gender perspective**: along with technical skills, we will keep an eye on how stereotypes shape who codes, and how we change that

## Opening the notebooks in Google Colab

We will use **Google Colab** to run the Python notebooks. It's a free platform
from Google that lets you write and run Python directly in your browser.

The notebooks are hosted on our GitHub repository:

> 🔗 [github.com/SheCodesPsy/SheCodesPsy-2026](https://github.com/SheCodesPsy/SheCodesPsy-2026)

### Step 1 — Open a notebook in Colab

There are three ways to do this — pick whichever works for you:

**Option A — Click the Open in Colab badge** 
Each notebook in the repository has an [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com) badge at the top. Just click it and the notebook will open directly in Colab.

**Option B — Quick URL trick**
After clicking on a `.ipynb` notebook file, go to your  browser's address bar and replace `github.com` with `githubtocolab.com` and press Enter.

**Option C — From inside Colab**
Go to [colab.research.google.com](https://colab.research.google.com), and click the **GitHub** tab. Then, paste the repository URL and press Enter, and click the notebook you want to open.

### Step 2 — Save a copy to your Google Drive ⚠️

Once the notebook is open in Colab:
1. Click **File** in the top menu.
2. Select **Save a copy in Drive**.
3. A new tab will open with *your personal copy* of the notebook, saved
   in a folder called `Colab Notebooks` in your Google Drive.
4. Always work from this copy — not the original link.

> Do this **every time** you start working with a new notebook. Without it, any changes you make will be lost the moment you close the tab.

---

## A few things to know about Colab

- **Sessions reset.** If you close the tab or leave it idle for ~30 minutes,
  Colab forgets everything — all variables are gone. Use **Runtime → Run all**
  to re-run the notebook from the top.
- **Run cells in order.** Each cell often depends on the ones above it.
  If you get an unexpected error, check that you haven't skipped any cells.
- **Your code is saved, your outputs are not.** After a reset, just re-run
  the cells to get your outputs back.

## Optional — Installing PsychoPy

**PsychoPy** is the software used by psychologists and neuroscientists to run experiments. It needs to be installed locally on your computer — it cannot run inside Google Colab.

> ⚠️ **This is not mandatory.** We will not do exercises with PsychoPy during the workshop. However, if you'd like to try running the experiment yourself while we walk through the live demo, here's how to install it. 
> ⏱️ Installation takes around **15–30 minutes**. If you'd like to have it ready, do this before the workshop day.

### Windows
1. Go to [psychopy.org/download.html](https://www.psychopy.org/download.html).
2. Under **Standalone PsychoPy**, click the **Windows** download link.
3. Run the `.exe` installer and follow the on-screen instructions.
4. Open **PsychoPy** from the Start menu to confirm it works.

### macOS
1. Go to [psychopy.org/download.html](https://www.psychopy.org/download.html).
2. Under **Standalone PsychoPy**, click the **macOS** download link.
3. Open the `.dmg` file and drag PsychoPy into your **Applications** folder.
4. Open PsychoPy from Applications.
